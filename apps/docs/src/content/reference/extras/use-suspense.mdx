The hook `useSuspense` is used to to get the `suspended`-state of the closest
`<Suspense>` boundary. For a complete implementation example, have a look at the
[`<Suspense>` component](/docs/reference/extras/suspense).

## Usage

### Suspend an Async Resource

Suspending a resource is simply triggered by using top-level await:

```svelte
<script>
  import { T } from '@threlte/core'
  import { useTexture } from '@threlte/extras'

  const texture = await useTexture('/texture.png')
</script>

<T.MeshBasicMaterial map={texture} />
```

### Get Suspended State

The hook can be used to get the `suspended`-state of the closest `<Suspense>` boundary.

```svelte
<script>
  const { suspended } = useSuspense()

  $inspect(suspended.current)
</script>
```
