A hook that sets the css cursor property according to the hover state of a mesh, so that you can give the user visual feedback.

If a context is present, the cursor property will be set on the DOM element of the renderer, otherwise it will be set on the body element.

<Example path="extras/use-cursor" />

## Examples

### Simple Usage

Provide arguments to determine the cursor style. The defaults are`'pointer'` for `onPointerOver`. `useCursor` returns event handlers that you can use to set the hovering state:

```svelte
<script lang="ts">
  import { T } from '@threlte/core'
  import { useCursor } from '@threlte/extras'

  // Set the cursor to 'grab' if the pointer is
  // hovering over the mesh and to 'crosshair'
  // if the pointer is outside the mesh
  const { onPointerEnter, onPointerLeave } = useCursor(() => 'grab')
</script>

<T.Mesh onpointerenter={onPointerEnter}>
  <T.BoxGeometry />
  <T.MeshBasicMaterial />
</T.Mesh>
```

### Change the Cursor Style

Provide svelte stores to change the cursor style also while hovering:

```svelte
<script lang="ts">
  import { T } from '@threlte/core'
  import { useCursor } from '@threlte/extras'

  let cursor = $state('grab')

  const { onPointerEnter, onPointerLeave } = useCursor(() => cursor)

  // somewhere in your application …
  cursor = 'grabbing'
</script>

<T.Mesh onpointerenter={onPointerEnter}>
  <T.BoxGeometry />
  <T.MeshBasicMaterial />
<T.Mesh>
```
